# CRA Frontend Design Rules

## Brand
- Palette: navy (#0A1628), cream (#F2F0EC), gold (#C9922A), electric blue (#1E6FFF)
- Fonts: Cormorant Garamond (headings), DM Sans (body)
- Tone: high-end, authoritative, cinematic

## Code Standards
- Vanilla HTML/CSS/JS only — no frameworks
- GSAP + ScrollTrigger for all animations
- Mobile-first responsive design
- All animations respect prefers-reduced-motion
- Custom cursor on desktop, hidden on touch devices

## Design Philosophy
- Bold typography, generous whitespace, asymmetric layouts
- Text reveals on scroll (fade up, stagger children)
- Parallax depth on hero and CTA background images
- Scroll snap (proximity) on major sections
- Curtain transitions — sections stack with increasing z-index
- Film grain overlay at low opacity for texture
- No generic AI aesthetics — no Inter, no purple gradients, no cookie-cutter cards

## Animation Defaults
- Reveal: y:60, opacity:0, duration:1, ease:power3.out, start:top 85%
- Stagger: 0.1–0.15s between siblings
- Parallax scrub: background moves at 50% scroll speed
- All transitions use cubic-bezier or GSAP easing, never linear

## File Structure
- index.html (homepage)
- services.html, about.html, contact.html, faq.html
- /css/style.css
- /js/main.js
- /images/ for all assets
