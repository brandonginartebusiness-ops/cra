# Competitor Feature Matrix

## Feature comparison

| Feature | How Common | Impact |
|---------|-----------|--------|
| Free consultation/inspection | Universal | Table stakes |
| No fee unless you get paid | Universal | Table stakes |
| Blog with regular content | ~80% of top firms | High SEO value |
| City-specific landing pages | ~65% | Critical for local rankings |
| Case studies with dollar amounts | ~50% | Strong conversion driver |
| Team bio pages with photos | ~60% | Trust builder |
| Video testimonials | ~20% | **Major gap to exploit** |
| Virtual inspection capability | Stellar only | Differentiator |
| Client portal/tracking | 2 firms only | Differentiator |
| Interactive tools/calculators | **None** | **Wide open opportunity** |
| Multi-language (Spanish/Creole) | ~25% | Essential for Miami |
| Live chat | ~10% | Underutilized |

## Gaps CRA can own

1. **Interactive claim calculator** — no competitor offers this
2. **Full bilingual website** (English + Spanish) — only Platinum PA does trilingual
3. **Video content** — very few produce it
4. **Client portal** — only 2 firms have this
5. **Miami Lakes / NW Miami-Dade hyper-local content** — completely uncontested territory
