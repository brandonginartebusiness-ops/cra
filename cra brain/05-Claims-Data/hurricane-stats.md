# Hurricane Damage Statistics — Florida

Sources: [FL OIR Hurricane Milton](https://floir.gov/home/hurricane-milton) · [FL OIR Hurricane Helene](https://floir.gov/home/hurricane-helene) · [FL OIR Hurricane Ian](https://floir.gov/home/ian) · [FL OIR Stability Report Jan 2026](https://floir.gov/docs-sf/default-source/property-and-casualty/stability-unit-reports/january-2026-isu-report-final.pdf)

## Hurricane Milton (2024)

| Metric | Value |
|--------|-------|
| Insured losses | **$5.62 billion** |
| Claims filed | **300,662+** |
| Context | Most costly single event since Ian |

## Hurricane Helene (2024)

| Metric | Value |
|--------|-------|
| Insured losses (FL only) | **$1.99 billion** |
| Claims filed | **135,505+** |

## Hurricane Ian (2022)

| Metric | Value |
|--------|-------|
| FL OIR-reported insured losses | **$21.4 billion** |
| Industry-wide estimates | $50–65 billion |
| Claims filed | **776,941** |
| Context | The benchmark event — still the most costly recent FL hurricane |

## Hurricane Idalia (2023)

| Metric | Value |
|--------|-------|
| Insured losses | $309.5 million |
| OIR exam findings | **8 of 10 insurers** guilty of claims-handling misconduct |
| Fines levied | **$2.075 million** |
